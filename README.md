# Facturación Kun-tu

App HTML pública para facturación, presupuestos y control financiero.

## Versión pública sin datos

Esta versión no contiene clientes, productos, facturas, gastos, proveedores, datos fiscales ni backups reales.

Los datos se guardan únicamente en el navegador mediante almacenamiento local. Para conservarlos, usa la opción de exportar backup JSON desde la app.

## Seguridad

No subas archivos `.json`, PDFs de facturas, Excels, datos fiscales o backups reales a este repositorio público.
